Day 3 — Chart Pattern Intelligence
① Pattern Detector (5 patterns)
  • Golden Cross (SMA50>SMA200, 5d window)
  • Death Cross (SMA50<SMA200, 5d window)
  • Volume Breakout (20d high + vol≥1.5x)
  • RSI Divergence (price LL + RSI HL, 14d)
  • Support Bounce (within 2% support, 5d)
② Backtester
  • History: 730 days lookback
  • Success: +2% within 15 trading days
  • Min events: 3 (else returns None)
  • Vol filter: exclude if vol < 0.8x avg
③ Pattern Explainer
  • 3 sentences · max 80 words
  • Rank: 0.3×strength + 0.7×success_rate
